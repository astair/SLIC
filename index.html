Site and Ligation Independent Cloning (SLIC)

The SLIC cloning methodxx is based on annealing of single-stranded complementary overhangs on the target vector and a PCR-generated insert. Single-stranded overhangs can be generated by using the 3'->5'-exonuclease activity of the T4 DNA polymerase.

Further information on the cloning procedure can be found on this EMBL page and in the original paperxx.

Melting Temperature Calculations

We implemented two differnt methods for calculation of the Tm:

(1) Wallace rule:

For less than 14 nucleotides, we actually use the Marmur formulaxx:

Tm = 4(G + C) + 2(A + T)

For 14 or more nucleotides, the Wallace rulexx is used:

Tm = 64.9 + 41(G + C - 16.4) / (A + T + G + C)

where G, C , A and T are the occurrences of the respective bases in the sequence. Both equations assume the standard conditions of 200 nM primer, 50 mM Na+, and pH 7.0.

(2) Base stacking:

This is a thermodynamical approach to approximate the Tm. The underlying script is essentially a JavaScript version of the PHP script used herexx,zz.
